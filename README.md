# Briefly

Briefly는 실시간으로 수집되는 다양한 RSS 뉴스 피드를 크롤링하고, AI 기술을 통해 핵심 내용을 3줄 요약하여 직관적인 다크 테마 웹 대시보드로 제공하는 오픈소스 AI 뉴스 요약 프로젝트입니다.

---

## 핵심 가치
* **정보의 효율적인 압축**: IT/Tech, 정치, 국제 분야 등 방대한 뉴스 기사 정보 중에서 가장 알맹이가 되는 요점만 한글로 빠르게 습득할 수 있도록 지원합니다.
* **완전한 자동화 파이프라인**: 1시간마다 주기적으로 최신 기사를 수집 및 요약하여 대시보드 화면을 스스로 갱신하도록 설계되었습니다.
* **직관적인 레이아웃**: 복잡하고 긴 줄글 텍스트를 배제하고, 깔끔한 다크 테마 카드 형식의 그리드 UI를 채택하여 뉴스 가독성을 극대화합니다.
